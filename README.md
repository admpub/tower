# Tower

Recompile your web server automatically when file changed.

[![Build Status](https://travis-ci.org/shaoshing/tower.png?branch=master)](https://travis-ci.org/shaoshing/tower)

## Install
```bash
go install github.com/shaoshing/tower
cd your/project
tower init
```
